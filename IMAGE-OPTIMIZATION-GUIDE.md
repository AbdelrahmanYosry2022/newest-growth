## üìã Image Optimization Best Practices

### File Format Recommendations:
- **Photos:** `.jpg` (optimized, 80-85% quality)
- **Logos/Graphics:** `.png` (with transparency if needed)
- **Icons:** `.svg` (scalable vector graphics)

### Compression Tools:
- TinyPNG (https://tinypng.com/)
- ImageOptim (Mac)
- Squoosh (https://squoosh.app/)

### Naming Convention:
- Use descriptive names: `growth-roots-factory-exterior.jpg`
- No spaces, use hyphens
- Include dimensions if multiple versions: `logo-400x200.png`
